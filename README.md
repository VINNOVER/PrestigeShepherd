# Prestige Shepherd

## Installation
This is a static HTML, CSS, and JavaScript website. No installation is required. Simply open the `index.html` file in a web browser to view the website.

## Usage
The website showcases the Prestige Shepherd dog breeding business. Users can:

- View information about the available dogs for adoption
- Browse a gallery of images related to the breeding and training process
- Learn about the business's commitments and features
- Contact the business to inquire about adopting a dog or scheduling a visit

## API
This website does not provide any API functionality.
